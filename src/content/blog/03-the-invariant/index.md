# The Invariant

人工智能时代，无论技术怎么发展，有太多东西不会变化。身体、健康、心情、睡眠、运动、社交——这些事情底层是 invariant，是操作系统级别的存在，不是可以被优化掉的东西。

---

身体、睡眠、运动、心情——这些不是要优化的事物，而是底层的操作系统。AI 改变的是 OS 层和 UI 层，而 Kernel 指向的东西不变：存在感、连接感、意义感。这些才是技术无法替代的部分，也是我们真正在寻找的东西。
